Profile Related

  • addshow.php uses stylesheet.css, dbAccess.php (addWatched()), addShowToWatched.php
    • an interface for user to add a show to a list (eg, "Watched") on their profile
    • linked to by plus buttons on profileshows.php
    • adds selected show to watchedList table in database
  • profilebooks.php uses stylesheet.css, dbAccess.php (getReadByUser, getReadingByUser, getFavoriteBooksByUser, getBookInfo)
    • a view of user's profile, where user's book lists can be seen
    • user can update lists using plus buttons
    • linked to an addbook.php file from plus buttons
    • displays elements from readList, readingList, and favoriteBooksList tables
    • will soon truncate titles and authors to fit in the list elements
  • profileshows.php uses stylesheet.css, dbAccess.php (getShowInfo, getEpsListByDate, getEpInfo)
    • a view of user's profile, where user's show lists can be seen
    • user can update lists using plus buttons
    • displays elements from watchedList, watchingList, and favoriteShowsList tables
    • linked to addshow.php from plus buttons
    • will soon truncate titles and creators to fit in the list elements
  • likes.php

    • may soon allow user to navigate to liked comment in ep/ch page by clicking it
  • addbookfavorites.php uses stylesheet.css, dbAccess.php (getBooksList, getBookInfo), addBookToFavorites.php
